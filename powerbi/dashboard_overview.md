# HR Analytics Dashboard - Power BI Implementation

## Dashboard Overview
Interactive Power BI dashboard analyzing HR metrics for 1,400+ employees with focus on attrition patterns and workforce insights.

## Key Metrics Visualized
- Total Employees: 1,400+
- Overall Attrition Rate: 16.12%
- Average Employee Age: 37 years
- Average Salary: $78,500

## Critical Insights
- R&D department shows highest attrition at 56%
- 24% of technical employees experience attrition
- Gender-based attrition: Female 14.8% vs Male 17%

## Dashboard Components
- Executive summary cards
- Department-wise attrition analysis
- Gender-based retention comparison
- Technical background impact assessment
- Salary distribution analysis
- Performance metrics visualization
